# Eclipse

A simple button component library with various styles and variants.

## Features

- **Primary, Secondary, Success, Danger, and Warning** button variants
- **Outline buttons** for a lighter visual style
- **Size variants** (Small, Medium, Large)
- **Disabled state** support
- **Hover and active states** for better interactivity
- Modern, clean design with smooth transitions

## Usage

Open `index.html` in your browser to see all button styles.

### Button Classes

| Class | Description |
|-------|-------------|
| `.btn` | Base button class (required) |
| `.btn-primary` | Blue primary button |
| `.btn-secondary` | Gray secondary button |
| `.btn-success` | Green success button |
| `.btn-danger` | Red danger button |
| `.btn-warning` | Orange warning button |
| `.btn-outline-*` | Outline variant of any button |
| `.btn-sm` | Small button size |
| `.btn-lg` | Large button size |

### Example

```html
<button class="btn btn-primary">Click Me</button>
<button class="btn btn-outline-danger btn-lg">Large Outline</button>
```
